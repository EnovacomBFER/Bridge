### Téléchargement et usage

Cet implementation guide contient un package mis à disposition dans cette page et permettant de valider les instances de ressources par rapport aux profils qu'il contient.

Pour rappel, cet implementation guide dérive de FR Core. Il est donc également possible de valider les instances de ressources par rapport au package mis à disposition sur <a href="https://hl7.fr/ig/fhir/core/downloads.html">l'implementation Guide FR Core</a>.
